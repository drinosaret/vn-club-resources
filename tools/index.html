    🧰 Tools for Reading VNs¶

    🧠 Text Hookers¶

    • Textractor (Chenx211 Fork) — Modern fork of the classic texthooker. Best for most modern Windows VNs.
    • Agent — Script-based texthooker used for cases where normal text-hookers won't work. Many scripts for emulated games (especially N#nt#n#o Sw#tc# games) are available.
    • ITHVNR — Replaced by textractor, but supposedly may work better with certain older games.
    • LunaTranslator — More bloated than all of the other options, but may work when the others fail and does have legitimately useful features. Apparently controversial for stealing code from Agent and Textractor, but must be included as a more modern texthooker. You better turn off the MTL.

    📖 Dictionaries¶

    • Yomitan — The browser based dictionary extension that everyone loves.
    • JL — The hover dictionary program made specifically for VN readers.
